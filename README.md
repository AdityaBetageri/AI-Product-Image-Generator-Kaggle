# AI Product Image Generator (Kaggle Project)

Kaggle competition project that converts product descriptions into AI-generated product images using **Qwen prompt engineering** and **Stable Diffusion**.

## 🏆 Achievement

* Achieved **Rank 68** on Kaggle competition leaderboard

## 🚀 Features

* Converts text descriptions into optimized image prompts
* Generates realistic product images
* Uses LLM-based prompt engineering
* Batch image generation for datasets
* Submission-ready CSV generation
* Interactive Gradio UI

## 🛠 Tech Stack

* Python
* PyTorch
* Transformers
* Diffusers
* Stable Diffusion
* Gradio
* Pandas

## 📌 Workflow

1. Input product description
2. Generate enhanced prompt using Qwen
3. Create product image using Stable Diffusion
4. Save outputs and submission file

## 📂 Files

* `ai_product_image_generator.ipynb` → Main notebook
* `final_score - Sheet1.csv` → Final results / scores
